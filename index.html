HTML Reference

Table of semantic HTML tags

Tags Names Explanations
<body> Body The <body> HTML element represents the content of an HTML document. There can be only one <body> element in a document. MDN
<header> Header The <header> HTML element represents introductory content, typically a group of introductory or navigational aids. MDN
<main> Main The <main> HTML element represents the dominant content of the <body> of a document. MDN
<section> Section The <section> HTML element represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it. MDN
<article> Article The <article> HTML element represents a self-contained composition in a document, page, or site, which is intended to be independently distributable or reusable. MDN
<footer> Footer The <footer> HTML element represents a footer for its nearest ancestor sectioning content or sectioning root element. MDN

CSS Reference

Table of a few CSS Properties

Properties Explanations Values
font-family The font-family CSS property specifies a prioritized list of one or more font family names and/or generic family names for the selected element. MDN It takes font names like: Georgia, serif, and so on.
background-image The background-image CSS property sets one or more background images on an element. MDN url("../../media/examples/lizard.png" and ...
color The color CSS property sets the foreground color value of an element's text and text decorations, and sets the currentcolor value. MDN <named-color> values like: "yellow", <hex-color> values like: "#090", and <rgb()> values like: "rgb(34, 12, 64, 0.6)"
